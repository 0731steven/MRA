# 变更记录

本项目遵循面向交付的变更记录；正式版本号和发布标签由交付双方确认。

## Unreleased

### 新增

- 班级归档/恢复、班级码更新和成员移出操作。
- 任务截止时间、撤回、重新发布和归档生命周期。
- 教学包 Markdown 导出与打印/保存 PDF。
- 手机和桌面真实浏览器自动验收、Docker Compose 生产栈健康检查。
- 系统架构和企业交付验收文档。

### 改进

- 修复手机题库被宽公式撑出屏幕的问题。
- 答疑设置和历史会话在手机端改为抽屉，改善消息和表格窄屏排版。
- 页面切换自动回到顶部，修复标题被固定导航遮挡。
- 教学包页面使用正确的页面语义，CI Actions 升级到当前稳定运行时。

### 数据兼容

- 本次功能复用现有班级和任务状态字段，不新增数据库迁移。
- 归档、撤回和移出成员均保留既有任务与作答证据。
